# FileSystemKit Documentation

This directory contains design documents, implementation notes, and technical documentation for the FileSystemKit project.

## Design Documents

- **MULTI_LAYER_ARCHITECTURE.md** - Overview of the multi-layer architecture (compression, disk images, file systems)
- **PERFORMANCE_IMPROVEMENTS.md** - Performance optimization plans and implementation notes
- **HASH_CACHE_IMPLEMENTATION.md** - Documentation for the file hash cache feature
- **CONCURRENT_PROCESSING.md** - Architecture for concurrent file processing

## Snug Archive Format Documentation

- **SNUG_FILE_FORMAT.md** - Snug archive file format specification
- **SNUG_YAML_FORMAT.md** - Snug YAML manifest format documentation
- **SNUG_ARCHIVE_EVALUATION.md** - Evaluation and design decisions for the Snug archive format
- **SNUG_DEEP_NESTING.md** - Documentation on deep nesting support in Snug archives

## Performance and Scaling

- **SMALL_FILE_HASHING.md** - Hash algorithm recommendations for small files
- **MILLIONS_OF_FILES_HASHING.md** - Hash algorithm recommendations for millions of files
- **PERFORMANCE_SCALING_MILLIONS.md** - Comprehensive performance evaluation and scaling recommendations for millions of files

## Custom Storage Providers

- **CUSTOM_STORAGE_PROVIDERS.md** - Guide for implementing custom storage backends (CloudKit, iCloud Drive, S3, etc.)
- **SNUG_CLOUD_STORAGE.md** - Cloud storage support for Snug archives (general usage)
- **RETROBOX_CLOUD_STORAGE.md** - Retrobox project cloud storage integration guide

## Architecture

- **ARCHITECTURE_MIGRATION.md** - Architecture overview and migration guide for FileSystemKit foundation and RetroboxFS extension

## Technical Notes

- **TAR_DIRECTORY_STRUCTURE_GAP.md** - Notes on TAR directory structure handling
- **GITHUB_ACTIONS_MONITORING.md** - Quick reference for monitoring GitHub Actions workflows
- **HOOKS_SETUP.md** - Git hooks setup and configuration
